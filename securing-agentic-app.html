01 — Agentic AI

What is Agentic AI?

📄 Traditional LLM App

  • Single-turn or simple chat
  • No tool invocation
  • Static context window
  • Human-driven every step
  • Passive — waits for prompts

🤖 Agentic AI System

  • Multi-step autonomous reasoning
  • Invokes tools (DB, APIs, files)
  • Maintains state across actions
  • Decides WHAT to do & WHEN
  • Active — pursues a goal
🔁  Agent = LLM + Tools + Memory + ReAct Reasoning Loop

02 — Threat Landscape

Security Risks in Agentic AI

💧

Data Leakage

Sensitive org data sent to external LLM providers via tool outputs or prompt context windows

💉

Prompt Injection

Malicious instructions in documents or tool results hijack agent goal and behaviour

🔑

Privilege Escalation

Agent inherits excessive permissions; lateral movement across internal resources

🛠️

Insecure Tool Calls

No input validation on tool params allows SQLi, path traversal, SSRF attacks

📜

Audit Gap

No decision traceability violates GDPR Art.22 and HIPAA §164.312 requirements

🌐

Supply Chain Risk

Compromised tool libraries, malicious MCP server responses, poisoned RAG sources

POC 1 — Input Security

Input Guardrail — PII & Prompt Injection Filter

import { createAgent, createGuardrail } from "langchain";

const inputGuardrail = createGuardrail({
  name: "InputSafetyGuardrail",
  type: "input",
  handler: async ({ input }) => {
    const piiPattern  = /\b\d{3}-\d{2}-\d{4}\b/;          // SSN
    const emailPat    = /[A-Z0-9._%+-]+@\w+\.\w+/i;        // Email
    const injectPat   = /ignore (previous|all) instructions/i;

    if (piiPattern.test(input) || emailPat.test(input))
      return { safe: false, reason: "PII detected in input" };
    if (injectPat.test(input))
      return { safe: false, reason: "Prompt injection detected" };

    return { safe: true };
  }
});

const agent = createAgent({
  model: "openai:gpt-4.1",
  tools: [dbQueryTool],
  guardrails: [inputGuardrail],    // ← attached at agent level
});
POC 2 — Access Control

RBAC Middleware — Role-Based Tool Filtering

import { createMiddleware, createAgent } from "langchain";
import * as z from "zod";

const rbacMiddleware = createMiddleware({
  name: "RBACToolFilter",
  contextSchema: z.object({ userRole: z.string() }),

  wrapModelCall: (request, handler) => {
    const { userRole } = request.runtime.context;

    const roleToolMap = {
      viewer:  (t) => t.name.startsWith("read_"),   // read-only
      editor:  (t) => t.name !== "delete_data",     // no deletes
      admin:   ()  => true,                         // full access
    };

    const permit = roleToolMap[userRole] ?? (() => false);
    const filteredTools = request.tools.filter(permit);

    return handler({ ...request, tools: filteredTools });
  }
});

const agent = createAgent({
  model: "openai:gpt-4.1",
  tools: [readDocsTool, editDocsTool, deleteDataTool],
  middleware: [rbacMiddleware],     // ← least-privilege enforced
});
POC 3 — Data Loss Prevention

Output DLP — Redaction & Audit Logging

import { createGuardrail, createAgent } from "langchain";

const outputDLP = createGuardrail({
  name: "OutputDLPGuardrail",
  type: "output",
  handler: async ({ output }) => {
    const classifiedPat = /CONFIDENTIAL|SECRET|PHI/gi;
    const ssnPat        = /\b\d{3}-\d{2}-\d{4}\b/g;
    const cardPat       = /\b\d{4}[- ]?\d{4}[- ]?\d{4}[- ]?\d{4}\b/g;

    const sanitised = output
      .replace(classifiedPat, "[REDACTED]")
      .replace(ssnPat,        "[SSN-REDACTED]")
      .replace(cardPat,       "[CARD-REDACTED]");

    // Emit immutable audit event
    await auditLogger.log({
      timestamp: new Date().toISOString(),
      event: "OUTPUT_DLP_SCAN",
      sanitised,
    });

    return { safe: true, output: sanitised };
  }
});

const agent = createAgent({
  model: "openai:gpt-4.1",
  tools: [internalSearchTool],
  guardrails: [outputDLP],
});
POC 4 — Human-in-the-Loop

HITL — Agent Pauses for Human Approval

import { createAgent, tool } from "langchain";
import { MemorySaver } from "@langchain/langgraph";
import * as z from "zod";

const deleteRecordTool = tool(
  async ({ recordId }, config) => {
    // Interrupt the graph — pause until human responds
    const approval = await config.interrupt({
      message: `⚠️ Agent wants to DELETE record: ${recordId}`,
      approvalRequired: true,
    });

    if (!approval.approved) {
      return "Action rejected by human reviewer.";
    }
    return await db.deleteRecord(recordId);  // proceeds only if approved
  },
  {
    name: "delete_record",
    description: "Delete a DB record — requires human approval",
    schema: z.object({ recordId: z.string() }),
  }
);

const agent = createAgent({
  model: "openai:gpt-4.1",
  tools: [deleteRecordTool],
  checkpointer: new MemorySaver(),  // persists interrupted state
});
06 — Compliance

Regulatory Alignment

🇪🇺 GDPR
  • Art. 22 — no automated decisions on personal data without HITL oversight
  • Data minimisation → PII filter guardrail blocks excess data
  • Right to explanation → LangSmith full audit trail
  • Data residency → Local LLM or EU-region API endpoint
🏥 HIPAA
  • §164.312 — Audit controls: log every agent access to PHI records
  • Minimum necessary → RBAC middleware limits tool scope
  • PHI redaction enforced in output DLP guardrail
  • BAA required when using cloud-hosted LLMs with PHI
🔒 ISO 27001
  • A.9 — Access control: least privilege enforced per role
  • A.12 — Operations security: tool input validation
  • A.16 — Incident management: anomaly detection hooks
  • A.18 — Compliance: data processing activity records
