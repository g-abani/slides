Classification Decision Tree

Page Classification Decision Tree ├── Does it contain product pricing/packaging information? │ ├── Yes → **Pricing & Packaging Page** ├── Is it primarily a customer story with quantified results? │ ├── Yes → **Customer Success Story/Case Study** ├── Is it a form-based page for lead capture? │ ├── Yes → **Contact/Demo Request Page** ├── Does it focus on a single product with comprehensive details? │ ├── Yes → **Product Detail Page** │ └── Is it a sub-feature of a larger product? │ └── Yes → **Product Sub-Feature Page** ├── Is it educational content (blog, tutorial, guide)? │ ├── Yes → **Blog/Educational Content Page** ├── Does it aggregate multiple content types/resources? │ ├── Yes → **Resource Center Page** ├── Is it event/webinar specific with registration? │ ├── Yes → **Event/Webinar Detail Page** ├── Does it combine multiple products for a solution? │ ├── Yes → **Solution/Use Case Page** ├── Is it the main entry point for the site/brand? │ ├── Yes → **Homepage/Corporate Landing Page** └── Otherwise → **Support/FAQ Page** or **Integration/Partnership Page**

Primary Classification Criteria

Criteria Category Classification Options
Content Purpose Commercial (product sales) vs Educational (learning) vs Transactional (conversion)
User Journey Stage Awareness (broad solutions) vs Consideration (specific products) vs Decision (pricing, demos)
Content Depth Overview (high-level) vs Detail (feature-specific) vs Deep-dive (technical)
Interaction Level Static (information consumption) vs Interactive (forms, tools) vs Transactional (purchases, requests)
